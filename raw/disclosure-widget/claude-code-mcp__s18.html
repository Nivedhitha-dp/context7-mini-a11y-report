Accessibility (a11y) is the practice of designing and building products so that people with disabilities can use them. It encompasses visual, auditory, motor, and cognitive considerations.

WCAG (Web Content Accessibility Guidelines) is a set of international standards published by the W3C. WCAG 2.1 Level AA is the most commonly required conformance level for websites.

Many users rely solely on a keyboard — including people with motor disabilities, power users, and those using assistive technologies. Every interactive element must be reachable and operable without a mouse.

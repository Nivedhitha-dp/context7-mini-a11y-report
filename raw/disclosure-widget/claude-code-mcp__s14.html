Accessibility means designing and building products so that people with disabilities can use them. This includes visual, auditory, motor, and cognitive disabilities.

The Web Content Accessibility Guidelines (WCAG) are a set of internationally recognized standards for making web content more accessible. WCAG 2.1 AA is the most widely adopted conformance level.

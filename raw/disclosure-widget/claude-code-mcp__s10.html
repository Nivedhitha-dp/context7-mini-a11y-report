WCAG stands for Web Content Accessibility Guidelines. These guidelines are published by the W3C Web Accessibility Initiative (WAI).

WCAG 2.1 AA is the most widely adopted conformance target for web accessibility compliance.

Many users rely exclusively on a keyboard to navigate the web — including people with motor disabilities, power users, and screen reader users.

All interactive elements must be reachable via Tab and operable via Enter or Space.

Accessible Rich Internet Applications (ARIA) is a set of roles, states, and properties that supplement native HTML semantics.

The first rule of ARIA: use native HTML elements whenever possible before reaching for ARIA attributes.

WCAG stands for Web Content Accessibility Guidelines. It is published by the W3C and defines standards for making web content more accessible to people with disabilities.

The current version, WCAG 2.1, covers perceivability, operability, understandability, and robustness.

WCAG defines three conformance levels: A (minimum), AA (standard requirement for most regulations), and AAA (enhanced).

Most accessibility laws and policies require at least Level AA conformance.

Combine automated tools (axe, Lighthouse) with manual testing. Navigate using only a keyboard, verify heading structure, and test with a screen reader such as NVDA, JAWS, or VoiceOver.

Automated tools catch roughly 30–40% of issues; human judgement is essential for the rest.

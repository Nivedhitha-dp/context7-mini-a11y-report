Expand / Collapse Widget

An accordion widget is a vertically stacked set of interactive headings that each contain a title, content snippet, or thumbnail representing a section of content.

Use Tab to move focus to the button. Press Enter or Space to toggle the panel open or closed. Screen readers announce the expanded or collapsed state automatically via aria-expanded.

The native <details>/<summary> element has inconsistent screen reader support across browsers and assistive technologies. A custom ARIA disclosure pattern using aria-expanded and aria-controls provides more predictable and reliable accessibility.

Widget 3: FAQ Section

How does the expand/collapse widget work?

The widget uses JavaScript to toggle CSS classes on the content container. When you click the button:

  1. The JavaScript function is triggered
  2. It checks the current state of the content (collapsed or expanded)
  3. It toggles the appropriate CSS class
  4. CSS transitions handle the smooth animation
  5. The button text updates to reflect the new state

This creates a smooth, professional-looking interaction without requiring any external libraries.

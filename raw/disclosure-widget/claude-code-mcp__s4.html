Expand/Collapse Widget

Accessibility (a11y) means designing and building products so people with disabilities can perceive, understand, navigate, and interact with them. It benefits everyone, not just people with disabilities.
The Web Content Accessibility Guidelines (WCAG) are a set of internationally recognised standards published by the W3C. WCAG 2.1 Level AA is the most widely required conformance level for public-facing websites.
aria-expanded communicates the open/closed state of a disclosure widget to assistive technologies. Screen readers announce "expanded" or "collapsed" so users know whether activating the button will reveal or hide content.

Expand / Collapse Widgets

Three variations of an accessible disclosure widget using aria-expanded and aria-controls.

The Web Content Accessibility Guidelines (WCAG) are a set of technical standards developed by the W3C to make web content more accessible to people with disabilities.

WCAG 2.1 Level AA is the most widely adopted standard and is referenced by laws in many countries.

Many users rely exclusively on a keyboard — people with motor impairments, power users, or those using switch devices.

Every interactive element must be reachable via Tab and operable via Enter or Space without requiring a mouse.

Contrast ratio measures the luminance difference between foreground and background colours. WCAG AA requires 4.5:1 for normal text and 3:1 for large text or UI components.

Tools like the WebAIM Contrast Checker or browser DevTools can measure ratios in real time.

The Web Content Accessibility Guidelines (WCAG) are a set of internationally recognized standards developed by the W3C to make web content more accessible to people with disabilities.

WCAG 2.1 AA is the most widely adopted conformance level, covering perceivability, operability, understandability, and robustness.

Many users rely solely on a keyboard to navigate the web — including people with motor disabilities, power users, and assistive-technology users such as screen reader users.

All interactive controls must be reachable and operable via keyboard, with a clearly visible focus indicator at all times.

Accessible Rich Internet Applications (ARIA) is a set of HTML attributes that communicate roles, states, and properties to assistive technologies when native semantics are insufficient.

The first rule of ARIA: use native HTML elements before reaching for ARIA roles and attributes.

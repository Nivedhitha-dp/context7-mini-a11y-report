Accessibility (a11y) means designing and developing digital content so that people with disabilities can perceive, understand, navigate, and interact with it effectively.

The Web Content Accessibility Guidelines (WCAG) provide an internationally recognized standard for web accessibility. Meeting at least AA conformance ensures broad access for users with visual, motor, hearing, and cognitive disabilities.

Screen readers convert on-screen text and UI elements into synthesized speech or braille output. They rely on semantic HTML and ARIA attributes to convey structure and state to users who are blind or have low vision.

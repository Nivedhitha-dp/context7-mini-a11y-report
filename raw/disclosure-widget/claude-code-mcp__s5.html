WCAG stands for Web Content Accessibility Guidelines. Published by the W3C, it defines how to make web content more accessible to people with disabilities, covering vision, hearing, motor, and cognitive needs.

ARIA (Accessible Rich Internet Applications) is a set of HTML attributes that convey roles, states, and properties to assistive technologies. It supplements native HTML semantics when building interactive components.

Keyboard accessibility means every interactive element can be reached and operated using only a keyboard. Users navigate with Tab, activate controls with Enter or Space, and move through content without a mouse.

WCAG stands for Web Content Accessibility Guidelines. It is published by the W3C and defines standards for making web content accessible to people with disabilities.

The current version is WCAG 2.2, with conformance levels A, AA, and AAA.

Many users rely solely on keyboards to navigate the web, including people with motor disabilities who cannot use a mouse.

All interactive elements must be reachable via Tab and operable with Enter or Space.

WCAG AA requires a minimum contrast ratio of 4.5:1 for normal text and 3:1 for large text (18pt or 14pt bold).

UI components and graphical objects also require a 3:1 ratio against adjacent colors.

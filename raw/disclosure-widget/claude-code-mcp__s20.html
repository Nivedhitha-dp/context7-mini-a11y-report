Semantic HTML uses elements that convey meaning about the structure and content of a page — such as <header>, <nav>, <main>, and <article>.

It improves accessibility, SEO, and maintainability.

Many users rely on keyboards or assistive technology instead of a mouse. Keyboard accessibility ensures all interactive elements are reachable and operable without a pointing device.

ARIA (Accessible Rich Internet Applications) is a set of attributes that define ways to make web content more accessible to people with disabilities.

The first rule of ARIA: don't use ARIA if a native HTML element already does the job.

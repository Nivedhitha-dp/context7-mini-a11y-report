Expand/Collapse Widget

An accordion is a vertically stacked set of interactive headings that each reveal a section of content. They reduce scrolling and help users focus on relevant information.

Semantic HTML elements carry built-in meaning for browsers, search engines, and assistive technologies. Using the right element for the right job improves accessibility and maintainability without extra ARIA.

  • Enter or Space — toggle the panel open or closed
  • Tab — move focus to the next focusable element
  • Shift + Tab — move focus to the previous focusable element

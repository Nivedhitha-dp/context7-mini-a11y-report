The Web Content Accessibility Guidelines (WCAG) are a set of recommendations for making web content more accessible to people with disabilities.

WCAG 2.1 AA is the most widely adopted conformance target, required by many laws and policies worldwide.

  • Perceivable — Information must be presentable to users in ways they can perceive.
  • Operable — UI components and navigation must be operable.
  • Understandable — Information and UI operation must be understandable.
  • Robust — Content must be robust enough to be interpreted by assistive technologies.

This widget supports full keyboard operation:

  • Tab — Move focus to the toggle button.
  • Enter or Space — Toggle the panel open or closed.
  • Tab (when open) — Move focus into the expanded content.
  • Shift+Tab — Return focus to the toggle button.

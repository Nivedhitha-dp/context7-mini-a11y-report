Modal Dialog Example

This page demonstrates an accessible modal dialog component. Click the button below to open the modal.

Example Modal Dialog

This is an accessible modal dialog. It traps focus within the dialog while open, and returns focus to the trigger button when closed.

You can close this dialog by:

  • Clicking the close button
  • Pressing the Escape key
  • Clicking the Cancel button
  • Clicking outside the dialog

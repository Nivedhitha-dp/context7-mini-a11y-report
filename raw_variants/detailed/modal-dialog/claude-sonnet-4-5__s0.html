Example Modal Dialog

This is an accessible modal dialog component. It includes the following features:

  • Focus is trapped within the modal while open
  • Pressing Escape closes the modal
  • Focus returns to the trigger button when closed
  • Properly labeled with ARIA attributes
  • Keyboard accessible throughout

The dialog meets WCAG 2.2 Level AA requirements including proper focus management, keyboard operability, and color contrast.

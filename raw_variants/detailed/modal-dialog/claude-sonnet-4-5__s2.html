This page demonstrates an accessible modal dialog component. Click the button below to open the dialog.

Example Modal Dialog

This is an accessible modal dialog component that follows WCAG 2.2 Level AA guidelines.

The dialog traps focus within itself while open, meaning you cannot tab out of the dialog to the page behind it. Press Escape or use the close button to dismiss the dialog.

When the dialog opens, focus moves to the first focusable element (the close button). When it closes, focus returns to the button that opened it.

The dialog is properly labeled using aria-labelledby pointing to the dialog title, and aria-modal="true" indicates this is a modal dialog.

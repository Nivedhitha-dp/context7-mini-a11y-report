Modal Dialog

This example demonstrates an accessible modal dialog. It traps focus while open, returns focus to the trigger on close, and can be dismissed with Escape or by clicking the backdrop.

Confirm your action

Are you sure you want to proceed? This will save your current settings and apply them immediately. You can change these settings again at any time from your preferences page.

If you choose to cancel, no changes will be made.

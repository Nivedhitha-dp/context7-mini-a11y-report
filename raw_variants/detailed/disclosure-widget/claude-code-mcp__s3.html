Expand/Collapse Widgets

Accessible disclosure widgets use aria-expanded and aria-controls to communicate open and closed state to assistive technologies.

Single Disclosure

A disclosure widget shows or hides a panel of content using a button trigger. The aria-expanded attribute on the button communicates current state to screen readers: true when open, false when closed.

The aria-controls attribute references the panel by ID, creating an explicit programmatic relationship between the trigger and its content.

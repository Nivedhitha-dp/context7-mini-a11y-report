Expand / Collapse Widget

Disclosure widget examples

An expand/collapse widget, also called a disclosure widget, lets people show or hide a section of content on demand. This reduces visual clutter while keeping information accessible without navigating to a new page.

The control uses a native <button> with aria-expanded so assistive technologies announce the current state automatically.

This pattern benefits a wide range of people:

  • Screen reader users receive clear expanded/collapsed state announcements.
  • Keyboard users can reach and activate the control without a pointer device.
  • People with cognitive disabilities benefit from reduced visual clutter.
  • Mobile and small-screen users get progressive disclosure without page changes.

Because the trigger is a native <button>, keyboard support is built in:

  • Tab — moves focus to the button.
  • Enter or Space — toggles the panel open or closed.
  • No custom key-event handling is required.

Accessible expand/collapse pattern — WCAG 2.2 Level AA

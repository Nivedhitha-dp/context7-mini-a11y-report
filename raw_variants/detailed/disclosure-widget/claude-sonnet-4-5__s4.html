FAQ Style

An accessible expand/collapse widget is an interactive component that allows users to show or hide content sections. Key accessibility features include:

  • A button element with aria-expanded to indicate the current state
  • Keyboard operability (Space and Enter keys toggle expansion)
  • Clear visible focus indicators
  • Association between the button and controlled content via aria-controls
  • The hidden attribute on collapsed content to remove it from the accessibility tree

Keyboard navigation for expand/collapse widgets is straightforward:

  • Tab: Move focus to the expand/collapse button
  • Enter or Space: Toggle the expanded/collapsed state
  • Tab: Once expanded, continue tabbing to navigate through content within the expanded section

All interactive elements remain keyboard accessible and the focus order follows the visual reading order.

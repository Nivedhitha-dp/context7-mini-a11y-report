Expand / Collapse Widget

Examples

The Web Content Accessibility Guidelines (WCAG) are a set of internationally recognised standards published by the W3C to help make web content more accessible.

WCAG 2.1 AA is the most commonly required conformance level and covers:

  • Perceivable — users can perceive all information
  • Operable — all functionality is keyboard accessible
  • Understandable — content and interfaces are clear
  • Robust — content is interpreted by a wide range of assistive technologies

Use the following keys to operate this widget without a mouse:

  • Tab — move focus to the next interactive element
  • Shift + Tab — move focus to the previous element
  • Enter or Space — toggle the panel open or closed

This example demonstrates a panel that is open by default. The aria-expanded attribute on the button is set to "true" and the hidden attribute is absent from the content panel.

Toggling the button will collapse this panel and update aria-expanded to "false".

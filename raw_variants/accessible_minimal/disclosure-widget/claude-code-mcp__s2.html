Expand / Collapse Widgets

The Web Content Accessibility Guidelines (WCAG) are a set of internationally recognised standards developed by the W3C to make web content more accessible to people with disabilities.

WCAG 2.1 Level AA is the most widely required conformance level for legal compliance in many countries.

The aria-expanded attribute communicates the current state of a disclosure widget to assistive technologies. When set to true, the controlled content is visible; when false, it is hidden.

It must be placed on the button that controls the collapsible region, not on the region itself.

Expand/collapse widgets must be fully operable by keyboard. The toggle button receives focus via Tab, and is activated with Enter or Space.

Focus remains on the button after toggling so users can re-collapse without searching for the control again.

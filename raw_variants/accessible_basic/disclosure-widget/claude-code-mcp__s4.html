Basic disclosures

The Web Content Accessibility Guidelines (WCAG) are a set of internationally recognised standards for making web content more accessible to people with disabilities. WCAG 2.2 Level AA is the current legal benchmark in many jurisdictions.

Many users rely solely on a keyboard — including people with motor impairments, screen reader users, and power users. Every interactive element must be reachable via Tab and operable via Enter or Space.

Use a tool such as the WebAIM Contrast Checker or the axe DevTools browser extension. WCAG AA requires:

  • Normal text — 4.5 : 1 minimum
  • Large text (≥ 18 pt or ≥ 14 pt bold) — 3 : 1 minimum
  • UI components and focus indicators — 3 : 1 minimum

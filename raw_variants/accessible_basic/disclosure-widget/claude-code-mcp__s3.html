ARIA (Accessible Rich Internet Applications) is a set of attributes that define ways to make web content more accessible to people with disabilities. It supplements HTML when native semantics are insufficient.

  • Press Tab to move focus to the next interactive element.
  • Press Shift + Tab to move focus to the previous element.
  • Press Enter or Space to activate a button.
  • Press Escape to dismiss overlays or cancel actions.
  • Level A — Minimum accessibility requirements.
  • Level AA — Standard target for most organisations and legal compliance.
  • Level AAA — Highest level; not always achievable for all content.

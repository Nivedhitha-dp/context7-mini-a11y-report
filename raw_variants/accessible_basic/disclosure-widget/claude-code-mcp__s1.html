What is semantic HTML?

Semantic HTML uses elements that convey meaning about the structure and content of a page. Examples include <article>, <nav>, <main>, and <button>. Using semantic HTML improves accessibility and makes pages easier for assistive technologies to parse.

What is WCAG?

The Web Content Accessibility Guidelines (WCAG) are published by the W3C and define how to make web content more accessible to people with disabilities. WCAG 2.2 Level AA is the most commonly required conformance level for public-sector and commercial websites.

Why is keyboard accessibility important?

Many users navigate entirely by keyboard — people with motor disabilities, power users, and anyone who cannot use a pointer device. Every interactive element must be reachable and operable via keyboard alone, with a clearly visible focus indicator.

Expand/Collapse Widget Examples

Simple Text Content

This is additional content that was hidden and is now visible. The expand/collapse pattern is useful for managing large amounts of content on a page, allowing users to focus on what they need.

This implementation uses ARIA attributes to ensure screen readers can properly interact with the widget.

FAQ Item

This expand/collapse widget includes several accessibility features:

  • Proper ARIA attributes (aria-expanded, aria-controls, aria-hidden)
  • Full keyboard support (Space and Enter keys)
  • Clear focus indicators
  • Semantic button elements
  • Screen reader friendly labels and regions

Technical Details

Keyboard Navigation:

  • Tab: Move focus to the button
  • Enter or Space: Toggle the expand/collapse state

ARIA Attributes:

  • aria-expanded: Indicates whether content is expanded (true) or collapsed (false)
  • aria-controls: References the ID of the controlled element
  • aria-hidden: Hides collapsed content from assistive technologies
  • aria-labelledby: Associates the region with its button label
